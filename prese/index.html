WebApp? WebApplication?

  • Anwendung (App) die über einen in das OS integrierten Browser läuft
  • ohne Installation
  • Daten werden auf einem Webserver gespeichert
  • Zugriff erfolgt über Internet

Vorteile

  • Keine Installation notwendig
  • Funktioniert auf allen gängigen Betriebssystemen
  • schnelle Veröffentlichung und Aktualisierung
  • Unabhängikeit von Stores
  • Offline-Speicherung von Daten mit HTML5 möglich
  • ---
  • Keine Push-Nachrichten

Nachteile

  • Keine Installation notwendig
  • Funktioniert auf allen gängigen Betriebssystemen
  • schnelle Veröffentlichung und Aktualisierung
  • Unabhängikeit von Stores
  • Offline-Speicherung von Daten mit HTML5 möglich
  • ---
  • Keine Push-Nachrichten
